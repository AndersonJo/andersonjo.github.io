# Installation

### Installing Docker

가상 컴퓨터를 위해서 사용합니다.

{% highlight bash %}
sudo apt-get update
wget -qO- https://get.docker.com/ | sh
{% endhighlight %}

### Installing Chef Development Toolkit

[Development Toolkit Download][dev-toolkit-url]
{% highlight bash %}
sudo dpkg -i chefdk_0.9.0-1_amd64.deb
{% endhighlight %}
